
### 超级淘工具箱

2017-11-15	Version v0.1α,Build 0010
2017-11-1x	Version v1.0β,Build 0110



功能简介：
	1.每天自动上传普通小票（5元）
	2.每 20 分钟摇一次红包
	3.合作商家小票点击上传（ToDo）


	
使用说明：
	1.首次使用请修改 conf/my.ini 中手机号（m）和密码（p）的值
	2.使用期间遇到问题，请将 logs/debug 文件夹内的日志发送给白楊
	3.请勿将 my.ini 文件发给任何人 × 3（重要的话说三遍）
	4.小票图片在 receipts 文件夹中，可自行添加
	5.合作商家小票在 receipts/recommend 文件夹中（ToDo）
